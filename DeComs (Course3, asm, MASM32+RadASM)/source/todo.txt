v.1.1
+ В конце работы последний файл закрывается и не виден.
+ Автоматическое разворачивание на весь экран в начале обработки и восстановление в конце.
+ При запуске параметры грузятся автоматически.
+ Нельзя начинать обработку без корректной инициализации параметров (проверка только на считывание из файла, но не на содержимое).
+ Пустой путь по умолчанию заменяется на текущий (путь к программе).
+ Оптимизация внутренних многоитерационных циклов (для БОЛЬШИХ файлов должно работать побыстрее).
+ В диалоге замены добавлена кнопка "Отмена" для прерывания работы. Больше ничего не будет обрабатываться и текущий файл не будет изменен.
+ Автоматически добавляется "\" в конец пути при необходимости (раньше бы ничего не работало).
+ Перестройка файла автоматической заменой "LF" на "CR/LF" {в памяти}. Измененные файлы сохраняются уже в формате "CR/LF".
+ Исправлена ошибка "терминального нуля" + еще куча не смертельных, но противных ошибок и недочетов (или перечетов, смотря как посмотреть :)
+ Динамическое определение типа конкретного файла ("LF" vs. "CR/LF").

+ Уменьшен (значительно) используемый размер стека.
+ Диалог с настройками.
+ Есть поддержка нескольких шаблонов многострочных комментариев.
+ В конце выводится список измененных файлов.

v.1.3
