Формат запросов 
Назад  Вверх 

 
Каждый запрос выглядит, как вызов функции. В круглых скобках через запятую перечисляются параметры. Некоторые параметры могут не использоваться (параметр по-умолчанию). 
 
  

ВНИМАНИЕ

Регистр букв имеет значение, т.е. запрос AddCol(...) будет выполнен, а запрос aDDCOL(...) нет.
 
Первая колонка содержит названия запросов. В строке с названием в 3 колонке записана общая форма запроса, где цифрами обозначены номера параметров, перечисленных ниже, во втором столбце. Напротив параметров 2 столбца в 3 столбце расположены описания соответствующих параметров. Если параметр записан в виде [x] - он не является обязательным (в запросе Вывода если не указать параметр 4, то вывод будет осуществлен в новую таблицу; в запросе Изменение параметр 2 используется если изменяется заголовок поля: вызов ChgName а не ChgType).

тип Тип поля: i - числовое, s - строковое.
нач. значение Значение полей новых записей (ВНИМАНИЕ: учитывайте типы полей).
позиция Номер колонки для вставки (от 0). Если нужно вставить в конец укажите -1.

Удаление записи DelRow(1, 2)
позиция Номер удаляемой записи.

Сортировка Srt(1, 2)

номер поля Номер поля, в котором будут производится сравнения.
условие Условие сравнения1.
относит. значение Значение, с которым будет производится сравнение (ВНИМАНИЕ: учитывайте типы полей).
[номер таблицы] Если параметр указан, то вывод будет производится в указанную таблицу. Иначе будет создана новая таблица.

номер 1 Позиции обмениваемых полей или записей.
номер 2
Изменение Chg[Type | Name]( 1[ , 2] )
номер поля Номер изменяемого поля. При изменении типа поля поля меняет тип на противоположный (Число <-> Строка). Если переход к числовому типу, то все нечисловые данные поля будут заменены 0.
[заголовок поля] Если выбрано изменение заголовка, то этот параметр содержит новое значение.
 

1 - Возможные условия сравнения:

  • Вывод записей с совпадающим полем  ?=
  • Вывод записей с полем большим заданного  ?>
  • Вывод записей с полем меньшим заданного  ?<
  • Вывод записей с количеством повторов поля равным некоторому количеству  +=
  • Вывод записей с количеством повторов поля большим некоторого количества  +>
  • Вывод записей с количеством повторов поля меньшим некоторого количества  +<
Для того, чтобы хорошо разобраться в запросах лучше всего создавать запросы в Мастере запросов, а затем используя этот раздел справки анализировать результаты. Также хорошо бы править запросы вручную и смотреть на получившийся результат.
