Мастер диаграмм  
Назад  Вверх  Вперед

 
Мастер диаграмм помогает выбрать данные для построения диаграмм и тип диаграммы.  
 
 
 
Данные для диаграммы выбираются из таблиц (список Таблица) среди их полей (список под списком Таблица). Выбранные поля хранятся в списке Выбранные записи и исходные таблицы. Для добавления и удаления используется двойной щелчек по записи в обоих списках. Тип диаграммы выбирается в списке Тип диаграммы. Для столбчатых и круговых диаграмм можно установить аксонометрический режим (Флаг Строить диаграмму в объеме).  
 
Для перехода к диаграмме нажмите кнопку Принять.  
 
 
ВНИМАНИЕ  
1)   Диаграммы можно строить только по числовым данным. Если среди выбранных полей окажется нечисловое поле, то будет выдано сообщение об ошибке.  
2)   При построении отрицательные числа берутся по-модулю.  
 

Окно с диаграммой, появляющееся после нажатия кнопки Принять в Мастере диаграмм, содержит 3 основные части:  
   1)   Панель с тремя кнопками: сохранения изображения в BMP-файл, вызов окна настроек диаграммы (см. ниже) и кнопки возврата к Мастеру диаграмм.  
   2)   Изображение построенной диаграммы.  
   3)   Вертикальной полосы прокрутки, доступной только в режиме круговой диаграммы и предназначенной для изменения эллиптического сжатия круга (Для круговой диаграммы в 3D это похоже на изменение угла взгляда в вертикальной оси).  
 
 
 

 
Окно настроек диаграммы содержит все возможные настройки диаграмм, связанные с внешним видом, т.е. не меняют данных, по которым строится диаграмма.  
 
Первая закладка Цвета и текст содержит настройки основных цветов (градиент, текст, линии), а также свойства элементов (подписи и цвет). Цвета меняются системным выбором цвета, вызываемом щелчком по цветному прямоугольнику. Для изменения подписи элемента выберите его в списке, в поле ниже введите новый текст и нажмите [Enter]. Без нажатия [Enter] введеный текст не будет сохранен в отличие от цветов.  
 
 
 
 
Вторая закладка Размеры содержит настройки различных числовых данных. Параметр Коэффициент сжатия круговой диаграммы является аналогом полосы прокрутки в окне с диаграммой.  
 
 
 
Третья закладка Параметры меняет флаги, описание которых приведено под ними.  
 
 
 
ВНИМАНИЕ  
До нажатия кнопки Принять никакие изменения не вступают в силу. После нажатия вернуть старое значение можно лишь в ручную.
